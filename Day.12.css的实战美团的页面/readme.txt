几个问题：
    1.对于一个前端开发工程师来说，最核心的技能是什么
        css  js

    2,要按web三层开发
        结构层：HTML  给网站提供内容
        样式层：css布局（流动布局，浮动布局，层布局）  美化模型（盒子模型，字体和文本设置等等）
        行为层：JavaScript    实现用户和浏览器之间的交互

    3，复习前面的核心内容
        html: 什么样的标签做什么样的事，尽量使用语义化标签，最核心的就是div，一定要合理使用div
        CSS：
            选择器：
                id选择器
                class
                标签选择器  p{}
                后代选择器  .div p{}
                交集选择器  p.div{}
                并集选择器  p,div{}
            盒子模型：
                width+height
                padding
                border
                margin
                background
            其他
                ：文字相关的font···，文本相关的text···，列表设置相关的···                        
            流式布局：默认的布局方案   块级标签独占一行  行级元素并排显示  从左到右 从上到下
                缺点，行标签不能并排显示
            浮动布局：实现字围效果，目的就是让块级元素并排显示，半脱离文档流
                造成对父元素和兄弟元素的影响
            层布局：
                说白了就是定位，相对定位：相对与自身原本的位置   偏移量（top bottom left right）
                               绝对定位：相对与参考点的位置（没有设置就是参考body的位置）   偏移量（top bottom left right）
                                固定定位：相对于浏览器窗口的位置   偏移量（top···）

    4.页面分层思想
        不同的文件放到不同的文件夹下，html放在最外面
        一个网站的首页面往往都叫 index.html
            整体的分层
        样式分层：
            reset.css   重置标签的默认样式  针对整个网站
            common.css   设置网站的通用的样式  也是针对整个网站
            index.css    表示首页面的样式  只针对当前的页面
            其他的页面创建符合的css
